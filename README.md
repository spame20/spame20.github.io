# Helen Duque — Portfolio

Single-file portfolio site. No build step, no dependencies to install — `index.html` is the whole site (GSAP loads from a CDN).

## Deploy on GitHub Pages

1. Create a new repository on github.com named **`YOURUSERNAME.github.io`** (public).
2. Add these files to the repo (drag-and-drop on github.com, or commit from VS Code).
3. On GitHub: **Settings → Pages → Source: Deploy from a branch → main / root → Save.**
4. Site is live at `https://YOURUSERNAME.github.io` within a minute or two.

If the repo has any other name (e.g. `portfolio`), the URL becomes `https://YOURUSERNAME.github.io/portfolio` instead. A custom domain (e.g. helenduque.com) can be added later under Settings → Pages → Custom domain.

## Before sharing widely

- The Vault section plays videos from the Google Drive folder. It must be shared as **"Anyone with the link — Viewer"** or visitors can't play them.
- Files `hd-logo-mark.svg` / `hd-logo-mark-black.svg` are the standalone logo files (dark/light backgrounds) — not required by the site, kept for reuse (social avatars, CV, etc.).
